Tomazela Lab — Pacote (rotas corrigidas: /lab/page/1 e /lab/[slug])

Copie estas pastas/arquivos para a raiz do seu projeto:
- app/lab/
- lib/posts.js
- content/lab/
- public/lab/

Atualize (se necessário) o package.json com:
"dependencies": {
  "gray-matter": "^4.0.3",
  "remark": "^15.0.1",
  "remark-html": "^16.0.1"
}

Acesse:
- /lab -> redireciona para /lab/page/1
- /lab/page/1 (paginação)
- /lab/[slug] para cada post

Para publicar novo artigo:
1) Crie um .md em content/lab/ com frontmatter (title, date, excerpt, cover opcional)
2) Coloque a capa em public/lab/
3) Commit no GitHub
